度数分布の中央値

度数分布のレコードから中央値情報をプロパティへ出力します　

■ストリーム情報

入力 フォーマット Record;CSV
接続数 1
出力 フォーマット
説明 入力ストリームをそのまま出力します。

■コンポーネントプロパティ

名前 プロパティ型 マッピング 説明
名前フィールド string 入力＆出力 入力ストリームの名前として使用するフィールド名を指定します。
値フィールド string 入力＆出力 入力ストリームの値として使用するフィールド名を指定します。
名前 string 出力 中央値が属する名前が入ります。
中央値 double 出力 中央値が入ります。

タイプ パラメーター エラー処理フロー

コード		 説明
汎用 なし コンポーネントの入力ストリーム なし その他のエラー
